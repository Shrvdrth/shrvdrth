## 🧠 Featured Build — MultiMind

> A full-stack **multi-agent AI decision analysis platform**. You ask a question — three specialized AI agents (**Strategist**, **Risk Analyst**, **Engineer**) independently debate it across two structured rounds, a **Moderator** synthesizes it all into a final call with a confidence score, and the whole debate streams live to your browser in real time.

<div align="center">

| Layer | Stack |
|---|---|
| **Frontend** | React 19 · TypeScript |
| **Backend** | ASP.NET Core (.NET 10, C#) |
| **Data** | PostgreSQL 16 · EF Core 9 |
| **AI** | OpenRouter API (multi-model orchestration) |
| **Realtime** | Server-Sent Events (SSE) |
| **Auth** | JWT + refresh token rotation |
| **Infra** | Docker · Kubernetes · GitHub Actions CI/CD |

</div>
